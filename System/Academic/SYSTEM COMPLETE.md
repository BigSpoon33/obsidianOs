### VIEW Syntax (Dynamic Headings)
```markdown
# ðŸ“š `VIEW[{course-code}]` - `VIEW[{course-name}]`
```
- Meta-bind displays frontmatter values
- Updates in real-time as you type
- Works in edit and reading mode

### Quick Controls (Inline Editing)
```markdown
> **Status:** `INPUT[inlineSelect(option(not-started), option(in-progress)):status]`
> **Due:** `INPUT[date:due]`
> **Grade:** `INPUT[text:grade]`
```
- Edit properties without opening frontmatter
- Instant updates
- No syntax errors
